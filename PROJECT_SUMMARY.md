## 🚀 خطوات التشغيل

```bash
# 1️⃣ تشغيل MongoDB
mongod

# 2️⃣ تثبيت المكتبات
cd backend && npm install
cd ../frontend && npm install

# 3️⃣ ملء قاعدة البيانات
cd backend && npm run seed:quran

# 4️⃣ تشغيل Backend
npm run dev

# 5️⃣ تشغيل Frontend (في Terminal جديد)
cd frontend && npm run dev
```
